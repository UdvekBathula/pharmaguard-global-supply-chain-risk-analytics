| Column                | Data Type | Description                         |
| --------------------- | --------- | ----------------------------------- |
| Batch_ID              | String    | Unique shipment identifier          |
| Drug_Name             | String    | Name of the medicine                |
| Manufacturer_Country  | String    | Country where manufactured          |
| Destination_Country   | String    | Final delivery country              |
| Market_Price          | Float     | Standard market price               |
| Selling_Price         | Float     | Actual selling price                |
| Expected_Transit_Days | Integer   | Planned shipping duration           |
| Actual_Transit_Days   | Integer   | Actual shipping duration            |
| Temperature_Required  | Float     | Required storage temperature        |
| Temperature_Recorded  | Float     | Recorded shipment temperature       |
| Reporting_Source      | String    | WHO, Customs, Consumer, etc.        |
| Lab_Result            | String    | Authentic, Substandard, Counterfeit |
